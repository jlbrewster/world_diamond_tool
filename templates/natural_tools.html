Natural Diamond Turning Tools

Natural Diamond Turning Tools are the preferred tools for finishing carbide dies. For quick set ups and super finishes. These tools come with polished tables, not sawn. Westec Diamond Tool specializes in these diamond tools and can polish in any radii the stone will accept.

Natural Diamond Turing Tools come on a 3/8 shank standard, others upon request. These tools come with a 76 ° included angle and a .007 radius as standard. Part number example-W7607 Part number stamped on shank.

If you cut carbide dies you need diamond turning tools and may not know it!!
